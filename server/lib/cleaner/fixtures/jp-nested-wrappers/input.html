Rustの所有権入門

Rustの所有権システムはメモリ安全性を保証する革新的な仕組みです。ガベージコレクタなしでメモリ管理を行えます。

借用チェッカー

借用チェッカーはコンパイル時にエラーを検出する強力なツールです。実行時のオーバーヘッドはゼロです。

fn main() {
    let s1 = String::from("hello");
    let s2 = &s1;
    println!("{}", s2);
}

ライフタイムは参照の有効期間を指定するための仕組みです。コンパイラが自動的に推論する場合もあります。
