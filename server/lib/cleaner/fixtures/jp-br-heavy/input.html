TypeScript型システム入門






TypeScriptの型システムは構造的部分型を採用しています。これにより柔軟で安全な型チェックが可能になります。







ジェネリクス




ジェネリクスを使うことで再利用可能なコードを書けるようになります。型パラメータにより関数やクラスを汎用化できます。

function identity<T>(arg: T): T {
  return arg;
}

const result = identity<string>("hello");








条件型

条件型は高度な型レベルプログラミングを可能にする強力な機能です。extends キーワードを使って型の分岐を表現します。





type IsString<T> = T extends string ? "yes" : "no";
type A = IsString<string>;  // "yes"
type B = IsString<number>;  // "no"
